The label modes are :

  • Latest version : unused
  • Label (Label name) : The name of label in TFS
  • Commit (Commit number) : the commit ID in TFS
  • Date (MM/DD/YYYY) : the date to be checkout
  • Workspace (Workspace;Owner) : The workspace name <WORKSPACE>;<USER>@<DOMAIN>
for further information see MSDN site.
